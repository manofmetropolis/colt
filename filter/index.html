The Filter Method

Creates a new array with all the elements that pass the test implemented by the provided function.

const nums = [9, 8, 7, 6, 5, 4, 3, 2, 1,];
const odds = nums.filter(n => {
return n % 2 === 1;
})
// Create a new array called "odds" by taking each thing in "nums" and returning the ones that meet the criteria, which in this case is: "Is the remainder of the number divided by 2 equal to one (i.e., "Is that shit an odd number?")"
